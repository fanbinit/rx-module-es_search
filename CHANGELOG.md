# 1.0.0 (2026-06-25)

라이믹스(Rhymix) 게시판 검색을 ElasticSearch로 대체하는 모듈의 첫 정식 릴리즈입니다.

## 주요 기능

- 게시판 검색(제목/내용/통합/댓글/글쓴이/별명/사용자ID/등록일)을 ElasticSearch 결과로 대체, ES 연결 실패 시 DB 검색으로 자동 폴백
- 댓글 색인 및 검색 지원 (댓글 검색, 내 댓글 보기), 비밀글 댓글은 비밀댓글로 간주해 권한 없는 검색에서 제외
- 한국어 형태소 분석기(nori) 자동 적용으로 복합어/조사 결합형 검색 정확도 향상
- 문서/댓글 변경 시 자동 동기화 대기열 및 소프트 스케줄러(크론 불필요)
- 동기화 현황/로그 확인, 수동 동기화, 인덱스 초기화 등 관리자 화면 제공
- 비회원 검색 차단 옵션

## 버그 수정(prerelease)

- 숫자 키워드 검색 시 무관한 문서가 매칭되는 문제 수정
- 본인 문서 검색 및 필드 범위 검색 관련 버그 수정
- 비밀댓글 검색 권한을 사이트 관리자뿐 아니라 게시판별 매니저 권한 기준으로 수정

## 실행 환경

- 라이믹스(Rhymix) 2.1.3 이상
- PHP 8.1.0 이상
- ElasticSearch + 한국어 형태소 분석기 플러그인(analysis-nori)

## 추후 지원할 기능

- 라이믹스 비동기 작업 기능과 연동 (소프트 스케줄러 대체)
- 통합검색 지원 (현재는 게시판 단위 검색만 지원)
